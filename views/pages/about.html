Why I Built Architoct: An Anti-PR, Democratic Space for Real Architecture Talk

I'm building an alternative to the PR-driven architectural media landscape. No partnerships, no sponsored content, no PR agencies deciding what's 'good architecture.'

Why?

  • ✕ ArchDaily shouldn't be architecture's gatekeeper
  • ✕ Knowledge shouldn't be paywalled
  • ✕ Young architects shouldn't need Instagram followers to be heard
  • ✕ Good work shouldn't need PR budgets

What it is:

ANONYMOUS

Say what you really think about that famous project, toxic office, or controversial development. No real names = real honesty.

DEMOCRATIC

Good content rises based on architect votes, not PR budgets. A great project from a small town firm can get more visibility than a mediocre one from a big name.

PRACTICAL

Building a vendor/contractor database with real reviews. Everything from steel suppliers to waterproofing experts. Phone numbers, rates, reliability scores - all crowd-sourced.

RADICALLY FREE

Information wants to be free. No premium features, no sponsored content, no ads disguised as articles.
